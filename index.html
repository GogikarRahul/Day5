VISUAL STUDIO CODE

VS code

1.Visual studio code(VS code) is a light weight but powerfull source code editor developed by Microsoft.

2.It supports a weide range of programming languages and comes with features.

Features of VS code:

  • Syntax highlighting and Auto-completion for most languages.
  • Integrated terminal for running scripts for commands.
  • Version control integration for Git and other commands for running Scripts or commands.
  • Extentions Marketplaceto and functionality such as language-specific support,linters,debuggers,theme and more.
  • Customizable interface with user-defined settings and shortcuts.
  • Built-in Debugging for a seamless coding experience.
